Sensor node costs

Sensor node bill-of-materials

There are many different types of sensors that may be useful in agricultural IoT applications. A sample list is provided below.

Aside from the sensors, a sensor node needs to provide the following functionality and characteristics:

  • LoRaWAN communication capability (radio, protocol stack)
  • Microcontroller to read sensors, manage communications, and control power modes
  • Power source, typically battery-based for field nodes
  • Enclosure suitable for outdoor use (e.g., IP65/66 rated)
  • Connectors or interfaces for the sensors (wired or wireless)
  • Low power consumption to maximise battery life
